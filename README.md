# 📱 Web Tabanlı Haber Uygulaması

Kullanıcı deneyimini ön planda tutan, mobil cihazlar için optimize edilmiş dinamik bir web tabanlı haber uygulaması. Bu proje, verileri statik HTML içine yazmak yerine dışarıdan (JSON) çekerek modern web geliştirme mimarisini temel alır.

## ✨ Öne Çıkan Özellikler

* **Dinamik Veri Yönetimi:** Haber içeriklerinin statik kodlardan değil, `news.json` dosyası üzerinden asenkron olarak çekilmesi ve ekrana basılması.
* **PWA (Progressive Web App) Altyapısı:** `manifest.json` dosyası sayesinde desteklenen tarayıcılarda ana ekrana bir "mobil uygulama" gibi eklenebilme özelliği.
* **Mobil Uyumlu (Mobile-First) Tasarım:** Mobil cihaz ekranlarına tam uyumlu, akıcı ve modern kullanıcı arayüzü (UI).
* **Saf (Vanilla) JavaScript:** Hiçbir dış kütüphane kullanılmadan yazılmış, doğrudan DOM manipülasyonu ve veri işleme algoritmaları.

## 📂 Proje Yapısı
* `index.html` - Uygulamanın ana iskeleti ve arayüzü.
* `style.css` - Özelleştirilmiş tasarım ve stillendirmeler.
* `script.js` - JSON verisini işleyen ve arayüzü dinamik olarak güncelleyen JavaScript kodları.
* `news.json` - Haber başlıklarını, içeriklerini ve detaylarını tutan veri simülasyonu.
* `manifest.json` - Uygulamanın cihazlar tarafından yüklenebilir bir web uygulaması olarak tanınmasını sağlayan yapılandırma dosyası.

## 🛠️ Kullanılan Teknolojiler
* HTML5
* CSS3
* Vanilla JavaScript
* JSON (Veri Modelleme)
